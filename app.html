Linked List Implementation

In this excercise I implement a doubly-linked list, this is a basic and very useful
data structure.
To use the linked list simply type the data you want to add in the input form and select one of the options to add or delete elements at will
The Append Button will add elements at the end of the list
The Add To Button will add the element at a specific position
The Pop Button will remove the element at the end of the list
The Remove Button will remove whatever element in the list , if it exists
